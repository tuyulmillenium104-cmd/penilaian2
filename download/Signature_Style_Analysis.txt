================================================================================
                    APA ITU CIRI KHAS (SIGNATURE STYLE)?
================================================================================

Ciri khas adalah elemen-elemen UNIK yang konsisten muncul di setiap konten Anda,
membuatnya recognizable sebagai "tulisan Anda" bukan AI generik.

┌─────────────────────────────────────────────────────────────────────────────┐
│                     KOMPONEN CIRI KHAS                                      │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ 1. VOICE SIGNATURE (Gaya Bicara)                                           │
│    ─────────────────────────────                                            │
│    - Kata-kata favorit yang selalu digunakan                               │
│    - Cara memulai kalimat                                                  │
│    - Cara mengakhiri paragraf                                               │
│    - Tone khas (sarkastik, enthusiastic, analytical, casual)               │
│                                                                             │
│ 2. STRUCTURAL SIGNATURE (Struktur Khas)                                    │
│    ────────────────────────────────────                                     │
│    - Pattern paragraf yang konsisten                                        │
│    - Cara menyusun argument                                                 │
│    - Transition style antar point                                           │
│    - Hook style yang selalu sama                                            │
│                                                                             │
│ 3. CONTENT SIGNATURE (Konten Khas)                                         │
│    ────────────────────────────────                                         │
│    - Perspective/angle yang unik                                            │
│    - Reference yang sering dipakai                                         │
│    - Metaphor/analogi signature                                             │
│    - Personal experience framework                                          │
│                                                                             │
│ 4. QUIRKS (Kebiasaan Unik)                                                 │
│    ──────────────────────────                                               │
│    - Catchphrase atau tagline                                               │
│    - Formatting khas                                                        │
│    - Emoji usage style                                                      │
│    - Grammar "mistakes" yang sengaja                                        │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

│                                                                             │
│ [ ] Analytical (thesis-driven, logical)                                     │
│ [ ] Casual/Conversational (relaxed, friendly)                              │
│ [ ] Data-Driven (numbers-focused, precise)                                  │
│ [ ] Storytelling (narrative, scenario-based)                               │

│ [ ] Technical (detail-oriented, professional)                              │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────── │
│                                                                             │
│ STEP 2: DEFINE SIGNATURE ELEMENTS                                          │
│ ──────────────────────────────────                                          │
│ Untuk setiap style, tentukan:                                               │
│                                                                             │
│ A. Hook Template (3-5 variasi)                                             │
│    ──────────────────────────                                               │
│    Style Analytical:                                                        │
│    - "[Subject] can't [action]."                                           │
│    - "The [problem] is real."                                              │
│    - "Here's what nobody talks about:"                                      │
│                                                                             │
│ B. Transition Phrases (3-5 variasi)                                        │
│    ────────────────────────────────                                         │
│    Style Analytical:                                                        │
│    - "This is where [X] comes in."                                         │
│    - "The solution?"                                                       │
│    - "Enter [X]."                                                           │
│                                                                             │
│ C. Closing Template (3-5 variasi)                                          │
│    ─────────────────────────────                                            │
│    Style Analytical:                                                        │
│    - "That's the [X]."                                                      │
│    - "[X] without [Y]. [Z] without [W]."                                   │
│    - "The [X] is clear."                                                    │
│                                                                             │
│ D. Signature Quirks (2-3 khusus Anda)                                      │
│    ─────────────────────────────────                                        │
│    - Specific contractions preference                                       │
│    - Punctuation style                                                      │
│    - Paragraph length preference                                            │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────── │
│                                                                             │
│ STEP 3: BUAT TEMPLATE MASTER                                               │
│ ─────────────────────────────                                              │
│ Contoh untuk Analytical Style:                                             │
│                                                                             │
│ ┌─────────────────────────────────────────────────────────────────────┐    │
│ │ [HOOK: Direct problem statement]                                     │    │
│ │                                                                       │    │
│ │ [CONTEXT: Why this matters (1-2 sentences)]                         │    │
│ │                                                                       │    │
│ │ [TRANSITION: "This is where [X] comes in."]                         │    │
│ │                                                                       │    │
│ │ [SOLUTION: What [X] does differently]                                │    │
│ │                                                                       │    │
│ │ [DETAILS: Technical explanation with signature phrases]              │    │
│ │                                                                       │    │
│ │ [THESIS: The bigger picture]                                         │    │
│ │                                                                       │    │
│ │ [CLOSING: "That's the [X]."]                                         │    │
│ └─────────────────────────────────────────────────────────────────────┘    │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────── │

│ STEP 4: PRACTICE KONSISTEN                                                  │
│ ─────────────────────────────                                               │
│ - Gunakan template yang sama untuk setiap konten                           │
│ - Tweak hanya untuk konteks spesifik                                       │
