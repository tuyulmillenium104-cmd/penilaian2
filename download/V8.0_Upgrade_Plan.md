### 5. EMOTION DERIVATION

| V7.0 FINAL | V8.0 |
|------------|------|
| Pre-defined emotions list | Calculated from multiple signals |
| Manual selection | Automatic calculation |
| Static | Dynamic per campaign |

```
V8.0 EMOTION CALCULATION:
──────────────────────────────────────────────────────────────────────────────
emotion_score = {
  market_component: 0.3 × derive_from_market(finance_data),
  topic_component: 0.3 × derive_from_topic(campaign_topic),
  gap_component: 0.2 × derive_from_gap(gap_analysis),
  timing_component: 0.2 × derive_from_timing(current_hour)
}

IF market_component indicates FEAR (Fear&Greed < 30):
  base_emotion = "reassurance"
  
IF gap_component indicates CONTROVERSY:
  emotion_modifier = "righteous_indignation"

FINAL_EMOTION = combine(base_emotion, emotion_modifier)
```

### 6. FR STRATEGY INTEGRATION

| V7.0 FINAL | V8.0 |
|------------|------|
| FR strategy AFTER content | FR strategy BEFORE generation |
| Generic reply bait | Calculated reply bait based on target accounts |

```
V8.0 FR CALCULATION:
──────────────────────────────────────────────────────────────────────────────
fr_target = calculate_fr_potential(leaderboard_top)

IF fr_target > 6.0:
  // Need influencer reply
  target_influencer_types = identify_target_accounts(campaign_topic)
  reply_bait_keywords = generate_reply_keywords(target_influencer_types)
  
  // Inject into generation
  generation_params.reply_bait = {
    keywords: reply_bait_keywords,
    question_format: TRUE,
    controversy_level: "moderate"
  }
```

### 7. SCORING PREDICTION

| V7.0 FINAL | V8.0 |
|------------|------|
| Manual scoring | LLM-assisted prediction |
| Post-generation only | Pre-submission optimization |

```
V8.0 SCORING FLOW:
──────────────────────────────────────────────────────────────────────────────
Step 1: Generate content
Step 2: LLM predicts scores for G1-G4, EP, TQ
Step 3: IF any predicted score < threshold:
          Identify weak areas
          LLM suggests improvements
          Apply improvements
          Re-predict scores
        ELSE:
          Proceed to submit

// This creates the iterative refinement loop automatically
```
