================================================================================
                    PHASE 1: CAMPAIGN RESEARCH
================================================================================

TUJUAN: Mengumpulkan semua informasi campaign sebelum membuat konten

--------------------------------------------------------------------------------
STEP 1.1: Fetch Campaign List
--------------------------------------------------------------------------------
API Endpoint: GET https://app.rally.fun/api/campaigns

Action:
- List semua campaign aktif
- Identifikasi campaign yang relevan
- Catat campaign address untuk detail lookup

Output:
- Campaign address
- Campaign title
- Basic info (token, rewards, dates)

--------------------------------------------------------------------------------
STEP 1.2: Fetch Campaign Detail
--------------------------------------------------------------------------------
API Endpoint: GET https://app.rally.fun/api/campaigns/{address}

Data yang harus di-extract:

┌─────────────────────────────────────────────────────────────────────────────┐
│ FIELD                  │ PURPOSE                                           │
├─────────────────────────────────────────────────────────────────────────────┤
│ goal                   │ Main campaign brief - untuk Content Alignment     │
│ knowledgeBase          │ Detailed info - untuk Information Accuracy        │
│ rules                  │ Campaign rules - untuk Campaign Compliance        │
│ style                  │ Tone guidelines - untuk voice matching            │
│ alpha                  │ Distribution curve (1, 3, atau 8)                 │
│ missions[]             │ Mission requirements                              │
│ token                  │ Reward token info                                 │
│ campaignRewards        │ Total reward pool                                 │
└─────────────────────────────────────────────────────────────────────────────┘

Output Format:
┌─────────────────────────────────────────────────────────────────────────────┐
│ CAMPAIGN INFO:                                                              │
│ - Title: [campaign title]                                                   │
│ - Address: [contract address]                                               │
│ - Alpha: [distribution curve value]                                         │
│ - Total Reward: [amount + token]                                            │
│ - Duration: [start date] - [end date]                                       │
│ - Min Followers: [requirement]                                              │
└─────────────────────────────────────────────────────────────────────────────┘

--------------------------------------------------------------------------------
STEP 1.3: Fetch Leaderboard
--------------------------------------------------------------------------------
API Endpoint: GET https://app.rally.fun/api/leaderboard?campaignAddress={address}&limit=100

Data yang harus di-extract:
- Top 20 scores (rank, username, points)
- Convert points dari atto (÷10^18) ke display score
- Identifikasi target score untuk beat Top 10

Output Format:
┌─────────────────────────────────────────────────────────────────────────────┐
│ LEADERBOARD TARGET:                                                         │
│ - Rank 1 Score: [X.XX]                                                      │
│ - Rank 10 Score: [X.XX] ← TARGET TO BEAT                                    │
│ - Rank 100 Score: [X.XX]                                                    │
│ - Total Participants: [count]                                               │
└─────────────────────────────────────────────────────────────────────────────┘

                    KEY PRINCIPLES
================================================================================

1. RESEARCH DULU, BUAT KONTEN KEMUDIAN
