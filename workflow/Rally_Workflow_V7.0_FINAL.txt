--------------------------------------------------------------------------------
1.2 GATE MULTIPLIER FORMULA
--------------------------------------------------------------------------------

gate_pass = min(G1, G2, G3, G4) > 0  (SEMUA gate harus > 0)
g_star = avg(G1, G2, G3, G4)         (Rata-rata gate score)
M_gate = 1 + β × (g_star - 1)        (β = 0.5, Multiplier 0.5x - 1.5x)

GATE MULTIPLIER TABLE:

| Gate Scores | g_star | M_gate | Effect |
|-------------|--------|--------|--------|
| All 2/2 | 2.0 | 1.5x | MAXIMUM BONUS |
| All 1.5/2 | 1.5 | 1.25x | GOOD BONUS |
| All 1/2 | 1.0 | 1.0x | BASELINE |
| Mixed 1-2 | ~1.5 | ~1.25x | PARTIAL |
| Any 0/2 | FAIL | N/A | DISQUALIFIED |

★ CRITICAL: All gates 2/2 = 50% BONUS to entire score!

--------------------------------------------------------------------------------
1.3 FR CHEAT CODE (EXPLAINED)
--------------------------------------------------------------------------------

FR Formula: log(Sum of Followers of All Repliers + 1)

CALCULATION EXAMPLES:
| Scenario | Total Followers | FR Score | Equivalent |
|----------|-----------------|----------|------------|
| 1 reply from 100K account | 100,000 | log(100001) ≈ 5.0 | = 100,000 likes |
| 10 replies from 10K each | 100,000 | log(100001) ≈ 5.0 | = 100,000 RTs |
| 1 reply from 1M account | 1,000,000 | log(1000001) ≈ 6.0 | = 1,000,000 likes |
| 100 replies from 100 each | 10,000 | log(10001) ≈ 4.0 | = 10,000 likes |

★ KEY INSIGHT: 1 reply from influencer > 100 replies from small accounts!
