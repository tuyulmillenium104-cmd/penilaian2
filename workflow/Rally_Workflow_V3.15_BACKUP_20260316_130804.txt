★ NEW: PHASE 4.10 - HASHTAG OPTIMIZATION (V3.15)
--------------------------------------------------------------------------------

OBJECTIVE: Maximize hashtag benefit, avoid penalty

--------------------------------------------------------------------------------
THE HASHTAG REALITY (VERIFIED):
--------------------------------------------------------------------------------

| Hashtag Count | Engagement Impact | V3.15 Rule |
|---------------|-------------------|------------|
| 0 | Baseline | ✅ OK for hooks |
| 1 | **+100% (2x)** | ✅ OPTIMAL |
| 2 | +80% | ✅ GOOD |
| 3+ | **-21%** | ❌ AVOID |

--------------------------------------------------------------------------------
HASHTAG RULES:
--------------------------------------------------------------------------------

RULE 1: MAXIMUM 1-2 hashtags per tweet
RULE 2: NEVER put hashtag in hook tweet (Tweet 1)
RULE 3: Place hashtag at END of tweet
RULE 4: If campaign requires 3+ hashtags, split across tweets
RULE 5: Optimal hashtag length = 18 characters
RULE 6: Use trending hashtags only if relevant

--------------------------------------------------------------------------------
HASHTAG PLACEMENT GUIDE:
--------------------------------------------------------------------------------

| Tweet Position | Hashtag Rule | Reason |
|----------------|--------------|--------|
| Tweet 1 (Hook) | ❌ NO hashtag | Dilutes hook impact |
| Tweet 2-4 | ✅ 1 hashtag OK | Adds discoverability |
| Tweet 5-6 | ✅ 1 hashtag OK | Context reinforcement |
| Tweet 7 (CTA) | ✅ 1-2 hashtags OK | Final discovery boost |

--------------------------------------------------------------------------------
EXAMPLE:
--------------------------------------------------------------------------------

❌ WRONG:
```
#Bitcoin #Crypto #DeFi are all trending today!
[Content]
```
(3 hashtags = -21% penalty)

✅ RIGHT:
```
Bitcoin is having a moment.

Here's what's happening [content]

#Crypto
```
(1 hashtag = +100% boost)
