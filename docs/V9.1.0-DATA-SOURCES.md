## 🔧 TEKNIS IMPLEMENTASI

### Web Search API Implementation:

```javascript
// Menggunakan z-ai-web-dev-sdk
const ZAI = require('z-ai-web-dev-sdk').default;

async function performWebSearch(query, numResults = 5) {
  const zai = await ZAI.create();
  const searchResult = await zai.functions.invoke("web_search", {
    query: query,
    num: numResults
  });
  
  // Returns array of:
  // { url, name, snippet, host_name, rank, date, favicon }
  return searchResult;
}
```

## ⚠️ APA YANG TIDAK DIAMBIL?

| Data | Status | Kenapa | Alternatif |
|------|--------|--------|------------|
| Twitter/X real-time feed | ❌ | Butuh API key berbayar | Web search mencakup |
| Reddit discussions | ❌ | Butuh Reddit API | Web search mencakup |
| Telegram messages | ❌ | Butuh Bot token | Web search mencakup |
| Discord conversations | ❌ | Butuh Bot token | Web search mencakup |
| Live price feeds | ❌ | Butuh Price API | Market search mencakup |
| On-chain transactions | ❌ | Butuh RPC endpoint | Tidak relevan untuk konten |

**Catatan:** Web search API mencakup banyak sumber termasuk social media snippets dan forum discussions.
