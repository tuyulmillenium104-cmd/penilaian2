# RALLY WORKFLOW - SINGLE PROMPT FOR NEW CHAT

## 📋 COPY PROMPT INI KE CHAT BARU:

```
Baca workflow dari GitHub: https://github.com/tuyulmillenium104-cmd/penilaian2

Buat konten Rally dengan hook: "CODE RUNS, DISPUTES DON'T. ENTER INTERNET COURT"

Ikuti workflow V8.7.6 lengkap dengan scoring system:
- Gate Utama (4 gates, min 4/5)
- Gate Tambahan (2 gates, min 8/8)  
- Penilaian Internal (6 metrics, min 9/10)
- Overall Score min 9.0

Tampilkan FINAL SCORE CARD di akhir.
```

## BANNED ITEMS (JANGAN GUNAKAN)

Words: delve, leverage, realm, tapestry, paradigm, catalyst, cornerstone, pivotal, myriad, moreover, furthermore, groundbreaking, game-changer, cutting-edge, unprecedented, ecosystem, landscape, foster, harness, robust, seamless, innovative, transformative

Phrases: in the world of, picture this, imagine a world, lets dive in, at its core, in conclusion, it is important to note, moving forward, not only but also, a testament to, it's worth noting, in the digital age, with the rise of

AI Patterns: in this thread, here's what you need to know, let me break it down, the bottom line is, what does this mean for you, key takeaways, in summary

Template Hooks: unpopular opinion, hot take, thread alert, breaking, this is your sign, psa, reminder that, quick thread, important thread, nobody is talking about, story time

## CONTENT REQUIREMENTS

1. STRUCTURE:
   - Gunakan EXACT hook sebagai kalimat pertama
   - Buat 2-4 tweet (maks 280 char per tweet, pisahkan dengan blank line)
   - Akhiri dengan pertanyaan engaging

2. EMOTION (butuh 3+ tipe):
   - Fear: risk, danger, crisis, fail, lost, vanish
   - Curiosity: what if, why, how, gap, missing, problem
   - Surprise: unexpected, finally, breakthrough, shocking
   - Hope: opportunity, future, possible, solution
   - Pain: lost, failed, broke, destroyed, regret
   
   WAJIB sertakan minimal 1 BODY FEELING:
   - stomach dropped, heart racing, cold sweat, chest tightened, couldn't sleep, jaw dropped

3. CT ELEMENTS (butuh score 8+):
   - Question (?): +2
   - Reply bait (what do you think, thoughts?): +2
   - Engagement hook (what if, have you ever): +2
   - Personal (I, my, me): +1
   - FOMO (now, today, finally): +1
   - Controversy (wrong, problem, nobody): +1
   - Share-worthy (this is why, the truth): +1

4. HOOK STANDARDS:
   - JANGAN buka dengan: the, a, an, this is, there are, I think
   - GUNAKAN: number, question, action verb, bold statement, personal pain

## OUTPUT FORMAT

1. Tampilkan konten yang di-generate
2. Hitung semua score
3. Tampilkan SCORE CARD dalam format box
4. Jika ada score yang tidak pass, regenerate (max 3x)

## 📝 CARA PENGGUNAAN:

### Step 1: Buka Chat Baru
Mulai conversation baru dengan AI

### Step 2: Copy-Paste Prompt
Paste salah satu prompt di atas ke chat

### Step 3: AI akan Mengeksekusi
AI akan:
1. Generate konten berdasarkan hook
2. Hitung semua score
3. Tampilkan Score Card
4. Regenerate jika perlu
