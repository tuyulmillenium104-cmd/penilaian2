## Execution Flow

### Phase 1-4: Data Gathering (INPUT)
1. **Phase 0**: Fetch campaign data (title, goal, rules, style, missions)
2. **Phase 1**: Research - extract facts from knowledge base + web search
3. **Phase 2**: Leaderboard analysis - identify top competitors
4. **Phase 2B**: Competitor deep analysis via LLM

### Phase 5-10: Content Creation (PROCESS)
5. **Phase 3**: Gap identification - find unique angles
6. **Phase 4**: Strategy definition - hook type, emotion target, CTA type
7. **Phase 5**: Content generation via LLM (multi-version)
8. **Phase 6**: Banned items scanner
9. **Phase 6B**: Rewrite if violations found
10. **Phase 7**: Uniqueness validation vs competitors
11. **Phase 8**: Emotion injection via LLM
12. **Phase 9**: HES + Viral scoring
13. **Phase 9B**: Viral enhancement (max 2x loop)
14. **Phase 10**: Quality scoring & selection (LOCK POINT)

### Phase 11-14: Optimization (REFINE)
15. **Phase 11**: Micro-optimization (5 layers)
16. **Phase 12**: Content flow polish
17. **Phase 12B**: Gate simulation (16 gates)
18. **Phase 13**: Benchmark comparison
19. **Phase 13B**: Beat Top 20 strategy (max 2x loop)
20. **Phase 14**: Final emotion re-check
21. **Phase 14B**: Final content polish

### Phase 15-16: Delivery (OUTPUT)
22. **Phase 15**: Output generation (Q&A, tags)
23. **Phase 15B**: CT Maximizer (max 2x loop)
24. **Phase 16**: Export and delivery

## Regeneration Rules

- **Maximum 3 regenerations** per workflow execution
- **Failback phases**: When a phase fails, return to earlier phase:
  - Phase 6B fail → Phase 5 (regenerate content)
  - Phase 7 fail → Phase 5 (regenerate content)
  - Phase 9 fail → Phase 8 (re-inject emotion)
  - Phase 9B fail → Phase 8 (re-inject emotion)
  - Phase 10 fail → Phase 5 (regenerate content)
  - Phase 12B fail → Phase 5 (regenerate content)
  - Phase 13B fail → Phase 4 (new strategy)
  - Phase 14 fail → Phase 8 (re-inject emotion)
  - Phase 14B fail → Phase 14 (re-polish)
  - Phase 15B fail → Phase 5 (full regeneration with CT focus)

## Banned Items

### Words to AVOID:
delve, leverage, realm, tapestry, paradigm, catalyst, cornerstone, pivotal, myriad, moreover, furthermore, groundbreaking, game-changer, cutting-edge, unprecedented, ecosystem, landscape, foster, harness, robust, seamless, innovative, transformative

### Phrases to AVOID:
in the world of, picture this, imagine a world, lets dive in, at its core, in conclusion, it is important to note, moving forward, not only but also, a testament to, it's worth noting, at the heart of, in the digital age, with the rise of

### AI Patterns to AVOID:
in this thread, here's what you need to know, let me break it down, the bottom line is, what does this mean for you, key takeaways, in summary

### Template Hooks to AVOID:
unpopular opinion, hot take, thread alert, breaking, this is your sign, psa, reminder that, quick thread, important thread, drop everything, stop scrolling, hear me out, let me explain, nobody is talking about, story time

## Hook Standards

### Weak Openings (AVOID):
- Starting with "the", "a", "an"
- "This is...", "There are...", "I think..."
- "In the...", "Today...", "So...", "Well..."

### Power Hook Patterns:
1. **Number/Data**: "$50M vanished...", "400 million users..."
2. **Question**: "What happens when...", "Who decides..."
3. **Action Verb**: "Imagine this...", "Picture..."
4. **Bold Statement**: "Code executes perfectly. Justice doesn't."
5. **Contrarian**: "No one talks about...", "Wrong..."
6. **Personal Pain**: "I lost everything...", "I couldn't believe..."
7. **Urgency**: "Warning: Your funds...", "Alert..."

### Required Hook Elements (need at least 2):
- Curiosity (what if, why, how, secret, hidden)
- Tension (but, however, wrong, problem, crisis)
- Surprise (unexpected, finally, breakthrough)
- Relevance (you, your, today, now)

## CT (Call-to-Action) Standards

### Required Elements (minimum 8/10):
1. **Question** (?): 2 points
2. **Reply Bait** (what do you think, thoughts?, who else): 2 points
3. **Engagement Hook** (have you ever, what if, imagine if): 2 points
4. **Personal** (I, my, me, we, our): 1 point
5. **FOMO** (now, today, finally, before): 1 point
6. **Controversy** (wrong, problem, fail, nobody): 1 point
7. **Share-worthy** (this is why, here's what, the truth): 1 point

## How to Execute

### Option 1: Using Node.js Script
```bash
node scripts/rally-ai-workflow.js "Your hook here"
```

### Option 2: Direct AI Execution
When AI receives a request like:
> "Create Rally content with hook: 'Code Runs, Disputes Don't. Enter Internet Court'"

AI should:
1. Generate content following all standards above
2. Score the content using the scoring functions
3. Check all gates pass
4. If not passing, regenerate up to 3 times
5. Display final score card
6. Return final content
